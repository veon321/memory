# Memory Card Game

A fast, engaging, and fully interactive frontend Memory game designed to test and improve cognitive skills. Built with vanilla technologies, this application features real-time scoring, dynamic countdowns, immersive sound effects, and smooth animations.

## Live Demo

[(https://veon321.github.io/memory-game/)]

## Technologies Used

- **HTML5:** Semantic architecture to organize the game board and control elements.
- **CSS3:** Modern UI styling, including CSS Transitions for flipping animations and flex/grid systems for the card layout.
- **JavaScript (Vanilla JS):** Core game mechanics, asynchronous Fetch API for card data, dynamic DOM manipulation, HTML5 Audio API, and interval-based timing logic.
